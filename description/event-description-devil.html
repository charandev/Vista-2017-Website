Devils Hands


Run. Error. Debug. Fix. Run Again... If you relate to these words, then Devil’S Hands is the spot for you. A coding event which brings together participants from all around the colleges. In each round the participant would be presented with a problem where the problem revolves around the correct output and saving their company from the devil’s hands.
